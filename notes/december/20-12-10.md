==How to set up the user name/email==

**`git config --global user.name`**
    
    Example:
    **`git config --global user.name "sergioAlmeida"`**

**`git config --global user.email "exemple@gmail.com"`**




==Starting git from the very begininng in a directory==

**`mkdir todoapp`**
**`cd toapp`**
**`ls`**

**`touch index.html`**

**`git status`**
**`git add .`**
**`ls  -la`**
